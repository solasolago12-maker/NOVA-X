# Security Policy

## Reporting a Vulnerability

If you believe you have found a security vulnerability in NOVA-X, please report it responsibly by emailing the project maintainers at `security@nova-x.dev`.

Do not open a public issue for security vulnerabilities.

## Supported Versions

We currently support the latest release of NOVA-X. If a security issue affects older versions, include the version number in your report.

## Response Process

1. Acknowledge receipt within 48 hours.
2. Investigate the issue and propose a fix.
3. Coordinate disclosure if necessary.
4. Release a patched version and notify the reporter.
